STEP 1: OPEN UNITY
├─ Open Unity Hub
├─ Click "JumperPOC" project
└─ WAIT for "Importing..." progress bar to finish (bottom-right)
   (This will take 30-60 seconds - Unity is creating .meta files)

STEP 2: OPEN TEST RUNNER
├─ In Unity Editor top menu: Window → General → Test Runner
└─ A new "Test Runner" window will open

STEP 3: VIEW TESTS
├─ Click "EditMode" tab in Test Runner window
├─ You should see:
│  📁 Tests.EditMode
│    📁 Tests.EditMode
│      📄 PlayerAbilitiesTests (19 tests)
│      📄 PlayerStateTrackerTests (26 tests)
│      📄 PlayerInputHandlerTests (16 tests)
│      📄 PlayerMovementTests (23 tests)
│      📄 PlayerHealthTests (34 tests)
│      📄 PlayerAnimationControllerTests (29 tests)
└─ Total: 147 tests

STEP 4: RUN TESTS
├─ Click "Run All" button at top of Test Runner
├─ Wait ~10 seconds
└─ ALL tests should show ✅ GREEN CHECKMARKS

EXPECTED RESULT:
✅ 147 tests passed
❌ 0 tests failed


═══════════════════════════════════════════════════════════════════

IF YOU DON'T SEE TESTS:

1. Unity hasn't finished importing yet
   → Check bottom-right corner for "Importing..." message
   → Wait until it says "All packages up to date"
   → Try Assets → Refresh (or press Ctrl+R)

2. Check Console for errors
   → Window → General → Console
   → Look for RED error messages
   → Send me any errors you see

3. Still nothing?
   → Close Test Runner
   → Right-click Assets/Tests folder → Reimport
   → Wait 30 seconds
   → Open Test Runner again (Window → General → Test Runner)

═══════════════════════════════════════════════════════════════════

WHAT TO DO AFTER TESTS PASS:

✅ All Green?
   → Excellent! Commit to git:
     git add .
     git commit -m "Expand unit test suite - 147 tests passing"

❌ Some Red?
   → Click the failed test to see error
   → Send me the error message
   → I'll help you fix it
